# Zgfh Tap

A collection of Homebrew formulae for tools by [@zgfh](https://github.com/zgfh).

## Install

```bash
brew tap zgfh/tap
```

Or install a formula directly:

```bash
brew install zgfh/tap/<formula>
```

Or via `Brewfile`:

```ruby
tap "zgfh/tap"
brew "<formula>"
```
